# Manuscript Draft

This folder contains the paper-style LaTeX draft built from the current repository outputs.

## Main file

- `cocoa_volatility_manuscript.tex`

## Bundle folders

- `figures/`: local copy of generated manuscript figures
- `tables/`: local copy of generated output tables
- `references/`: local bibliography and citation note for portable compilation

## Bundle refresh

From the repository root:

```bash
python scripts/12_prepare_latex_bundle.py
```

This refreshes the local `paper/figures`, `paper/tables`, and `paper/references` folders so the manuscript is easier to move and compile outside the main repository.

## Build sequence

From the repository root:

```bash
cd paper
pdflatex cocoa_volatility_manuscript.tex
bibtex cocoa_volatility_manuscript
pdflatex cocoa_volatility_manuscript.tex
pdflatex cocoa_volatility_manuscript.tex
```

## VS Code without `latexmk`

This workspace now includes local VS Code settings in `.vscode/` that use:

- `pdflatex`
- `bibtex`
- `pdflatex`
- `pdflatex`

instead of `latexmk`, so Perl is not required.

If VS Code still tries `latexmk`, reload the window and run the LaTeX Workshop command:

```text
Build LaTeX project
```

or use the built-in task:

```text
Tasks: Run Build Task
```

which calls `paper/build_manuscript.ps1`.

The manuscript uses bibliography entries from `../references/cocoa_volatility.bib` and figures generated by the pipeline in `../outputs/figures/`.
The portable bundle version uses local copies in `paper/references/` and `paper/figures/`.
