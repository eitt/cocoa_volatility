# Manuscript Draft

This folder contains the paper-style LaTeX draft built from the current repository outputs.

## Main file

- `cocoa_volatility_manuscript.tex`

## Build sequence

From the repository root:

```bash
cd paper
pdflatex cocoa_volatility_manuscript.tex
bibtex cocoa_volatility_manuscript
pdflatex cocoa_volatility_manuscript.tex
pdflatex cocoa_volatility_manuscript.tex
```

The manuscript uses bibliography entries from `../references/cocoa_volatility.bib` and figures generated by the pipeline in `../outputs/figures/`.
